```java
// 参考评论区答案
// 切分成[0, i)是去重后的subArray，[i, n-1]是搜索的部分
// 1. 前两个不管是否重复都不管
// 2. 如果搜索的元素和i-2一样的话，说明[i-2, 搜索元素]部分是重复的，这个重复一定于2（因为搜索元素的index>=i）
// 这种时候就要继续搜索到下一个和i-2不一样的元素为止，并且进行替换
// 只要还是重复的，i就不会增加，继续停在该搜索位置
// 所以就能保证重复的元素一定最多只有两个
int i = 0;
for (int n : nums)
    if (i < 2 || n > nums[i-2])
        nums[i++] = n;
return i;
```
